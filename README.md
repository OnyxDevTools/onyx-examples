# onyx-examples
Onyx Database Integration Examples

## Example projects
- [python-django/](python-django/) — Django Notes REST API wired to Onyx Database via generated models; Swagger/OpenAPI playground served at the root URL with scripts to run locally.
- [typescript-express/](typescript-express/) — TypeScript Express task-tracking API fully backed by Onyx Database with Swagger UI at the root
- [kotlin-ktor/](kotlin-ktor/) — Kotlin Ktor contacts CRUD API using Onyx Database with env-based client setup and Swagger UI.
- [java-springboot/](java-springboot/) — Spring Boot 3 Customer CRUD API with Onyx Database client DI wiring, validation, and Swagger UI at `/swagger-ui.html`.
- [go-gin/](go-gin/) — Go Gin CRUD API backed by Onyx with generated client code, Swagger UI at the root, health check, and items endpoints.
